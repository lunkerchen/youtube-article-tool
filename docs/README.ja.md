# 📺 YouTube 記事変換ツール (YouTube to Article Tool)

このツールは、YouTube動画の内容を迅速に深く構造化された記事に変換するAI搭載ツールです。`yt-dlp`で字幕を抽出し、Google Gemini AIを用いて論理的に再構成することで、口語的な動画内容を高品質な書き言葉の知識ベースに変換します。

## ✨ 主な機能

- **並列変換**: 複数のYouTubeリンクを一度に入力でき、非同期I/Oにより複数の動画を同時に処理するため、変換効率が大幅に向上します。
- **深い構造化**: 単なる要約ではなく、AIが核心となるテーマを抽出し、「テーマブロック」形式で内容を再構成し、具体的な事例や金言を保持します。
- **多言語サポート**:
  - **出力言語**: 記事の出力言語を指定できます（中国語、繁体字中国語、英語、日本語、韓国語など）。
  - **UI言語**: 繁体字中国語、英語、日本語の3種類のUIインターフェース切り替えをサポートしています。
- **履歴管理**: 直近の変換履歴をローカルに保存し、クイックレビューや削除が可能です。
- **Markdown書き出し**: 生成されたMarkdownファイルをワンクリックでコピーまたはダウンロードでき、Obsidianなどの知識管理ツールに最適です。

## 🚀 クイックスタート

### 1. 環境準備
システムに以下の依存関係がインストールされていることを確認してください：
- **Python 3.9+**
- **yt-dlp** (コア字幕抽出ツール)
  ```bash
  pip install yt-dlp
  ```
- **必要な Python ライブラリ**
  ```bash
  pip install fastapi uvicorn httpx python-multipart
  ```

### 2. ツールの起動
プロジェクトディレクトリに移動し、起動スクリプトを実行します：
```bash
bash start.sh
```
起動後、ブラウザで `http://127.0.0.1:8080` を開いてください。

### 3. 使用フロー
1. インターフェースに **Gemini API Key** を入力します。
2. 記事の**出力目標言語**を選択します。
3. **YouTubeリンク**を1つまたは複数（1行に1つ）貼り付けます。
4. 「立即啟動並行轉換 (並列変換を開始)」をクリックします。

## 🛠️ 技術スタック

- **Backend**: FastAPI (Python) + asyncio
- **Frontend**: Tailwind CSS + Marked.js (Markdown rendering)
- **AI Model**: Google Gemini Flash
- **Data Extraction**: yt-dlp

## 📄 ファイル構造

- `main.py`: バックエンドのコアロジック。字幕抽出とAIプロンプトエンジニアリングを処理します。
- `templates/index.html`: フロントエンドのシングルページアプリケーション (SPA)。UIインタラクションとi18nを処理します。
- `start.sh`: サーバーを迅速に起動するためのスクリプト。
- `history.json`: ローカルの履歴保存ファイル。

## ⚠️ 注意事項
- **API Key**: このツールはサーバー側にAPI Keyを保存しません。利便性のためにフロントエンドの `localStorage` に一時的にキャッシュされます。
- **字幕への依存**: このツールは動画に字幕（自動生成字幕を含む）があることに依存しています。字幕が全くない動画は変換できません。
