# YouTube to Article Tool 📺 ➡️ 📝

A high-performance pipeline that transforms YouTube videos into high-quality, structured, and readable articles using **yt-dlp** and **Google Gemini LLM**.

## 🌐 Languages / 語言

- [English](./docs/en.md)
- [繁體中文](./docs/zh-TW.md)
- [日本語](./docs/jp.md)

---

### 🚀 Quick Start / 快速上手

1. **Clone**: `git clone https://github.com/lunkerchen/youtube-article-tool.git`
2. **Install**: `pip install -r requirements.txt`
3. **Run**: `./start.sh`
4. **Open**: `http://localhost:8000`

Check the detailed guides in the language folders above. / 請參閱上述語言文件以獲取詳細指南。
